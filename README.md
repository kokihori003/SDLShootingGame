# SDLShootingGame
 
C++、SDLを使用したシューティングゲームです。
VisualStudioでビルドするとプレイできます。

# ゲームのルール

WASDで移動、スペースキーで攻撃ができます。

10回攻撃するとリロードが行われます。

小惑星 : ダメージを受けます。攻撃で破壊できます。

レンチ : 回復します。攻撃で破壊できます。

長時間生存し、小惑星を破壊し、スパナを獲得することでスコアを獲得できます。
