# SDLShootingGame
 
C++、SDLを使用したシューティングゲームです。
VisualStudioでビルドするとプレイできます。  
C++でのゲーム開発を学習中で、学習内容のアウトプットのために開発しています。


# ゲームのルール

* WASDで移動、スペースキーで攻撃ができます。
* 10回攻撃するとリロードが行われます。
* 小惑星 : ダメージを受けます。攻撃で破壊できます。
* レンチ : 回復します。攻撃で破壊されます。
* 長時間生存し、小惑星を破壊し、レンチを獲得することでスコアを獲得できます。
